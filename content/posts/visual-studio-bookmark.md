## Shortcuts

Show *Bookmarks* window: Ctrl + k, Ctrl + w

Toggle bookmark of a code line: Ctrl + k, Ctrl k

Skip to the next bookmark: Ctrl + k, Ctrl + n

Skip to the previous bookmark: Ctrl + k, Ctrl + p
